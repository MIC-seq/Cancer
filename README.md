# Cancer

This repository contains codes necessary for analysis of "Dynamic cross-species single-cell transcriptomic mapping deciphers gut microbiota functional signatures and ecotypes driving immunotherapy responses in colorectal cancer". 

Please contact weiqinjiang@zju.edu.cn, yifeishen@zju.edu.cn and fangyumo@zju.edu.cn with any questions.



## Downloading the data

The microbial single-cell RNA-seq data, metagenomic sequencing data, and metabolomics data generated in this study have been deposited at Genome Sequence Archive under accession number “PRJCA065884”.

   - Seurat
   - dplyr

   - RColorBrewer
